# AR을 이용한 위치기반 타입캡슐
- 지도앱을 이용하여 타임캡슐을 나타낸다.
- 사용자는 자신이 갔던 오프라인 장소에 타임캡슐을 설정하여 개인이나 단체로 타임캡슐을 설정할 수 있다. 
- AR을 이용하여 오프라인 장소에서 이전에 설정해놓은 타임캡슐을 생동감있게 열어볼 수 있다.
- 캡슐의 위치는 현장에서 등록하고 등록된 기록으로 집에서도 수정이 가능하다.
- 캡슐의 잠금 기간 설정 기능 제공, 기간에 따라 캡슐의 이미지를 달리하여 보여준다.
- 자신의 캡슐을 히스토리 순으로 확인이 가능하다.
- 캡슐의 ♡♥



## 탐험 + SNS

- 자신의 Map을 갖고 모험을 떠난다.
- 자신의 시야에 들어온 Map의 정보를 얻게된다. 타임캡슐을 발견하게 된다.

- 온라인으로 자신의 Map 공유가 가능하여 다른 사용자와 소통이 가능하다. ( 공유 페이지 제공 )

## 추가 기능
- 친구등록된 친구나 오픈된 사용자의 Map과 타임캡슐을 볼 수 있다.
- 팔로우한 대상이나 자신의 캡슐 근처 지역을 지나갈때 ( AR확인해보세요 )알림 기능
